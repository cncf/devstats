This is a toolset to visualize GitHub archives using HA Postgres databases and Grafana dashboards.
Everything is open source so that it can be used by other CNCF and non-CNCF open source projects.
The only requirement is that project must be hosted on a public GitHub repository/repositories.
Project is deployed using Equinix bare metal Kubernetes nodes and deployed using a Helm chart.
It uses many more CNCF projects under the hood.
